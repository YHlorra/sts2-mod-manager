## Description (填在 Description 栏，支持 BBCode 格式)

[size=5][b]STS2 Mod Manager[/b][/size]

A modern, lightweight desktop application for managing your Slay the Spire 2 mods. No more manually copying files — install, enable/disable, and organize all your mods from one beautiful interface.

[url=https://github.com/ImogeneOctaviap794/sts2-mod-manager]GitHub Repository (Open Source)[/url]

[size=4][b]✨ Features[/b][/size]

[list]
[*][b]One-Click Install[/b] — Install mods from ZIP files or drag-and-drop directly into the manager
[*][b]Enable / Disable[/b] — Toggle mods on and off without deleting them
[*][b]Dependency Detection[/b] — Automatically detects missing dependencies and warns you before launch
[*][b]Smart Categorization[/b] — Mods are auto-classified as Framework, Gameplay, or Resource with color-coded tags
[*][b]Risk Signals[/b] — DLL, PCK, and dependency badges help you understand what each mod does at a glance
[*][b]Dual View Mode[/b] — Switch between a card grid view and a compact list view
[*][b]Sorting[/b] — Sort mods by name, dependency issues, gameplay impact, category, or file size
[*][b]Profile System[/b] — Save and load mod profiles to quickly switch between different mod setups
[*][b]Built-in Translation[/b] — Translate English mod names and descriptions to Chinese with one click (translations are saved locally)
[*][b]Save Manager[/b] — Backup, export, import, and restore your save files
[*][b]Log Viewer[/b] — View game logs and crash analysis directly in the manager
[*][b]Launch Game[/b] — Start Slay the Spire 2 directly from the manager with real-time game state tracking
[*][b]Batch Operations[/b] — Multi-select mods in list view for batch enable/disable/uninstall
[*][b]First-Use Onboarding[/b] — Step-by-step guide for new users
[/list]

[size=4][b]📥 Installation[/b][/size]

[b]Option 1: Installer (Recommended)[/b]
[list=1]
[*]Download [b]STS2-Mod-Manager_1.1.0_x64-setup.exe[/b] from the Files tab
[*]Run the installer — it will set up everything automatically
[*]Launch "STS2 Mod Manager" from your Start Menu or Desktop
[/list]

[b]Option 2: Portable[/b]
[list=1]
[*]Download [b]sts2-mod-manager.exe[/b] from the Files tab
[*]Place it anywhere and run — no installation needed
[/list]

[size=4][b]🚀 Getting Started[/b][/size]

[list=1]
[*]Launch the manager and select your Slay the Spire 2 game directory (e.g. [font=Courier]D:\SteamLibrary\steamapps\common\Slay the Spire 2[/font])
[*]The manager will automatically scan and display all installed mods
[*]Drag-and-drop any mod ZIP file to install, or click "Install MOD"
[*]Enable/disable mods with the toggle switch
[*]Click "Launch Game" to start playing!
[/list]

[size=4][b]⚙️ Requirements[/b][/size]

[list]
[*]Windows 10/11 (64-bit)
[*]Slay the Spire 2 installed via Steam
[*]WebView2 Runtime (included in installer, or pre-installed on Windows 11)
[/list]

[size=4][b]🔒 Privacy & Safety[/b][/size]

[list]
[*]Fully open source — [url=https://github.com/ImogeneOctaviap794/sts2-mod-manager]view the code on GitHub[/url]
[*]The only network request is an optional translation API call (MyMemory) when you click the Translate button
[*]No telemetry, no analytics, no accounts required
[/list]
