## What's New in v1.1.0

### 🛡️ MOD 风险信号
- 缺失依赖的 MOD 卡片顶部显示醒目红色横条，明确标出「缺失依赖，无法正常工作」
- 自动识别 MOD 分类：**框架前置**（被其他 MOD 依赖）、**玩法改动**（含 DLL 或影响玩法）、**资源类**
- 每种分类有独立颜色和图标，一眼看清能不能安全启用

### 🔗 依赖跳转
- 详情面板中的依赖项变为可点击链接，直接跳转到对应 MOD

### 🔀 智能排序
- 新增排序下拉：按名称 / 依赖问题优先 / 影响玩法优先 / 按分类 / 按大小

### 🖼️ 双视图模式
- 卡片网格视图（默认）和双栏列表视图随时切换
- 列表视图支持多选 + 批量操作

### 🧭 首次使用引导
- 未识别游戏目录时显示 3 步卡片引导流程：选目录 → 装 MOD → 启动验证
- 已选目录但无 MOD 时，第 1 步打勾，第 2 步高亮引导安装

### ⚠️ 操作确认
- 卸载、批量卸载、应用配置档案、还原备份等风险操作弹出确认对话框
- 卸载有依赖的 MOD 时额外警告哪些 MOD 会受影响

### 其他
- 默认窗口尺寸增大到 1400×900
- MOD 安装改为智能解压到子文件夹
- 集成 GitHub 快捷导航图标
- 配置档案功能整合到 MOD 管理页面
